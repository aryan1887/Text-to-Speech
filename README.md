# Text-to-Speech Translation
A simple **Text-to-Speech** web app that converts text input intp spoken audio by using the browser's **SpeechSynthesis API**.

## Features
1)Convert any text into speech
<br>
2)User can choose mulitple voices from the dropdown available in browser

## Technologies Used
1)HTML5
<br>
2)CSS3
<br>
3)JS
<br>
4)**SpeechSynthesis API** (Web Speech API)

## How to Use
1)Open "index.html" in your browser.
<br>
2)Write any text in the textarea field.
<br>
3)Select a voice from DropDown Menu
<br>
4)Click to Listen button to hear the text written in textarea.
